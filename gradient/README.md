# Pixel Gradient UI — 渐变色版

> 像素风 + 渐变色融合 | 前缀 `--pg-` / `.pg-`

完整文档请查看 **[根目录 README](../README.md)**

## 快速引入

```html
<svg xmlns="http://www.w3.org/2000/svg" style="display:none">
  <!-- 从 pixel-icons.svg 复制 symbol 到这里 -->
</svg>
<link rel="stylesheet" href="pixel-gradient.css">

<button class="pg-btn pg-btn-primary">按钮</button>
```

## 设计特点

- 渐变按钮 + 发光阴影
- 柔和圆角（4px ~ 24px）
- Inter 字体

**在线预览**: 打开本目录下的 [index.html](./index.html)
