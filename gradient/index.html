Icons 图标

如何使用 How To Use

第一步：引入文件

下载或引用以下文件到你的项目中：

<!-- 1. 引入图标 Sprite -->
<svg xmlns="http://www.w3.org/2000/svg" style="display:none">
  <!-- 复制 pixel-icons.svg 内容到这里 -->
</svg>

<!-- 2. 引入样式表 -->
<link rel="stylesheet" href="gradient/pixel-gradient.css">

第二步：使用组件

直接在 HTML 中使用 class 名称：

<!-- 按钮 -->
<button class="pg-btn pg-btn-primary">主要按钮</button>

<!-- 带图标的按钮 -->
<button class="pg-btn pg-btn-primary">
  <svg class="pg-icon"><use href="#pg-icon-search"></use></svg>
  搜索
</button>

<!-- 输入框 -->
<input class="pg-input" placeholder="请输入..." />

<!-- 卡片 -->
<div class="pg-card">
  <div class="pg-card-body">卡片内容</div>
</div>

<!-- 图标 -->
<svg class="pg-icon"><use href="#pg-icon-home"></use></svg>

第三步：自定义主题（可选）

通过 CSS 变量快速换色：

:root {
  --pg-primary-500: #667eea;     /* 主色 */
  --pg-radius-md: 12px;          /* 圆角 */
}

切换暗色主题

给 html 标签加 data-theme 属性即可切换暗色模式：

<html data-theme="dark">
