# Pixel Custom UI — 自定义版

> 全 CSS 变量驱动 | 前缀 `--pc-` / `.pc-`

完整文档请查看 **[根目录 README](../README.md)**

## 快速引入

```html
<svg xmlns="http://www.w3.org/2000/svg" style="display:none">
  <!-- 从 pixel-icons.svg 复制 symbol 到这里 -->
</svg>
<link rel="stylesheet" href="pixel-custom.css">

<button class="pc-btn pc-btn-primary">按钮</button>
```

## 一行换色

```css
:root { --pc-primary: #ff6b35; }
```

**在线预览**: 打开本目录下的 [index.html](./index.html) （含颜色定制器）
