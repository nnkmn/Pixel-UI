# Pixel Minimal UI — 极简风版

> 硬像素风 | 前缀 `--pm-` / `.pm-`

完整文档请查看 **[根目录 README](../README.md)**

## 快速引入

```html
<svg xmlns="http://www.w3.org/2000/svg" style="display:none">
  <!-- 从 pixel-icons.svg 复制 symbol 到这里 -->
</svg>
<link rel="stylesheet" href="pixel-minimal.css">

<button class="pm-btn pm-btn-primary">按钮</button>
```

## 设计特点

- 0 圆角 / 硬边框
- 方块硬阴影（无模糊）
- step-end 复古动画
- 等宽/像素字体
